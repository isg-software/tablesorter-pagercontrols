This example page holds the same table as examples-fullwidth-scroll.html, only it does not use the scrolling or fullwidth template, but it does use the tablewidth template.

If the window is narrower than the table, this results in a whole page that is scrollable in order to be able to see the whole table. But as you can see here, there's a problem: Since the pager buttons in tablewidth preset are aligned with the table, and if your window is narrower that the table, you can't even see the buttons for switching to the next pages without scrolling all the way to the right. And if you do so, you'll have to scroll back to the left in order to reach the buttons for switching to previous pages.

So this is clearly not a good configuration. For tables this wide which might overflow, the template scrolling is much better suited, which may optionally be combined with the fullwidth template as demonstrated in examples-fullwidth-scroll.html.
